Smart File Assistant

Smart File Assistant is a Streamlit app that lets you upload files (PDF, TXT, CSV) and ask questions about their content using Retrieval-Augmented Generation (RAG).
The app extracts text, creates embeddings, builds a vector database, and generates answers with context, related sections, and follow-up questions.

Features

Upload multiple files

Supports PDF, TXT, CSV

Automatic text extraction + chunking

Vector search using Chroma + FastEmbed

LLM answers based on retrieved context (RAG)

Shows relevant document snippets

Generates follow-up questions
